# Numerical Methods Toolkit

A desktop application built in **Python** that implements several numerical methods through an interactive graphical user interface (GUI). The application allows users to solve numerical analysis problems, visualize results, and explore mathematical algorithms in an intuitive way.

## Features

* Interactive GUI built with Tkinter
* Root finding using the Bisection Method
* Solving systems of linear equations using:

  * Jacobi Method
  * Gauss-Seidel Method
* Lagrange Interpolation with graph visualization
* Numerical Integration using the Trapezoidal Rule
* Dynamic mathematical expression input
* Error handling and input validation
* Convergence checking
* Automatic diagonal dominance detection
* Embedded graphs using Matplotlib

## Technologies Used

* Python
* Tkinter
* NumPy
* Matplotlib

## Numerical Methods Implemented

* Bisection Method
* Jacobi Method
* Gauss-Seidel Method
* Lagrange Interpolation
* Trapezoidal Rule

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Run:

```bash
python GUI.py
```
